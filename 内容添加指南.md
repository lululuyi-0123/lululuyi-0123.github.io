# 内容添加指南

本文档详细说明需要在哪些位置添加图片、视频和文本内容。

## 📁 文件夹结构

请确保以下文件夹已创建：
- `images/` - 主图片文件夹
- `images/projects/` - 项目截图文件夹
- `images/design/` - 平面设计作品文件夹
- `images/photography/` - 摄影作品文件夹
- `videos/` - 视频文件夹（可选）

## 🖼️ 图片添加位置

### 1. 个人简介页面 - 个人照片

**文件**: `about.html`  
**位置**: 第32行左右  
**需要添加**: `images/about-photo.jpg`  
**说明**: 你的个人照片，建议尺寸 800x1000px

```html
<img src="images/about-photo.jpg" alt="个人照片" class="about-photo">
```

---

### 2. 项目经历页面 - 项目截图

**文件**: `projects.html`  
**位置**: 每个 `project-card` 中的 `project-image` 部分

**需要添加的图片**:
- `images/projects/project-1.jpg` - 项目一截图
- `images/projects/project-2.jpg` - 项目二截图
- `images/projects/project-3.jpg` - 项目三截图
- 更多项目：复制 `project-card` 结构并添加新图片

**建议尺寸**: 1200x800px

**如何添加更多项目**:
1. 复制整个 `<div class="project-card">...</div>` 结构
2. 修改图片路径（如 `project-4.jpg`）
3. 修改项目标题、描述、日期等信息
4. 修改 `data-keywords` 属性以支持搜索

---

### 3. 平面视觉页面 - 设计作品

**文件**: `design.html`  
**位置**: 每个 `gallery-item` 中的 `gallery-image` 部分

**需要添加的图片**:
- `images/design/design-1.jpg` - 设计作品1
- `images/design/design-2.jpg` - 设计作品2
- `images/design/design-3.jpg` - 设计作品3
- `images/design/design-4.jpg` - 设计作品4
- `images/design/design-5.jpg` - 设计作品5
- `images/design/design-6.jpg` - 设计作品6
- 更多作品：复制 `gallery-item` 结构并添加新图片

**建议尺寸**: 1000x1000px（正方形）或根据作品比例

**如何添加更多作品**:
1. 复制整个 `<div class="gallery-item">...</div>` 结构
2. 修改图片路径（如 `design-7.jpg`）
3. 修改作品标题和年份
4. 修改 `data-keywords` 属性（用于搜索功能）

---

### 4. 摄影页面 - 摄影作品

**文件**: `photography.html`  
**位置**: 每个 `photo-item` 中的 `photo-image` 部分

**需要添加的图片**:
- `images/photography/photo-1.jpg` - 摄影作品1
- `images/photography/photo-2.jpg` - 摄影作品2
- `images/photography/photo-3.jpg` - 摄影作品3
- `images/photography/photo-4.jpg` - 摄影作品4
- `images/photography/photo-5.jpg` - 摄影作品5
- `images/photography/photo-6.jpg` - 摄影作品6
- 更多照片：复制 `photo-item` 结构并添加新图片

**建议尺寸**: 1200x900px（4:3比例）或根据照片比例

**如何添加更多照片**:
1. 复制整个 `<div class="photo-item">...</div>` 结构
2. 修改图片路径（如 `photo-7.jpg`）
3. 修改作品标题、拍摄地点、日期
4. 修改 `data-keywords` 属性（用于搜索功能）

## 🎥 视频添加位置

### 摄影页面 - 视频作品（可选）

**文件**: `photography.html`

**需要添加的视频**:

### 2. 项目经历页面 (`projects.html`)

对于每个项目卡片，需要编辑：

#### 项目标题
- **位置**: `<h2 class="project-title">项目名称一</h2>`
- **说明**: 修改为你的项目名称

#### 项目日期
- **位置**: `<p class="project-date">2024年</p>`
- **说明**: 修改为项目完成时间

#### 项目描述
- **位置**: `<p class="project-description">...</p>`
- **说明**: 详细描述项目背景、你的角色、使用的技术、项目成果等

#### 技术标签
- **位置**: `<span class="tag">技术标签1</span>`
- **说明**: 添加项目使用的技术或工具标签

#### 搜索关键词
- **位置**: `<div class="project-card" data-keywords="...">`
- **说明**: 添加关键词以便搜索功能使用（用空格分隔）

## 💡 提示

1. **图片优化**: 建议压缩图片以提升网站加载速度
2. **文件命名**: 使用小写字母和连字符，避免空格和特殊字符
3. **图片格式**: 建议使用 JPG 或 PNG 格式
4. **备份**: 修改前建议备份原始文件
5. **测试**: 添加内容后，在浏览器中测试所有功能是否正常
